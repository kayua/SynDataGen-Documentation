 84    =================================================

 86    The Synthetic Ocean AI library is designed for the generation of tabular data using generative models.
 87    It provides comprehensive GPU support, allowing for efficient processing, and can be used either as a
 88    framework or as a Python library, offering flexibility depending on the user's requirements.
 89
 90    The library supports data ingestion from various formats, including CSV and XLS, enabling seamless

143    Model Catalog:
144    --------------
145        1. Adversarial (GAN) [model_type='adversarial']
146
147            Implements an adversarial training algorithm, typically used in Generative Adversarial Networks (GANs).
148
149            This class performs adversarial training by utilizing a generator and a discriminator,
150            optimizing the generator to produce realistic data while training the discriminator to differentiate
151            between real and fake data.
152
153        2. Autoencoder [model_type='autoencoder']
154
155            Implements a  AutoEncoder model for generating synthetic data.
156
157            This class implements an Autoencoder model by inheriting from the VanillaEncoder and VanillaDecoder classes.
158            It constructs an autoencoder architecture by combining both an encoder and a decoder with customizable
159            hyperparameters. The autoencoder is typically used for tasks such as dimensionality reduction, feature learning,
160            and denoising.
161
162        3. Variational Autoencoder [model_type='variational']
163
164            Implements a Variational AutoEncoder (VAE) model for generating synthetic data.
165
166            The model includes an encoder and a decoder for encoding input data and reconstructing
167            it from a learned latent space. During training, it computes both the reconstruction loss
168            and the KL divergence loss. The trained decoder can be used to generate synthetic data.
169
170        4. Wasserstein GAN [model_type='wasserstein']
171
172            A Wasserstein Generative Adversarial Network (Wasserstein GAN) model.
173
174            This class represents a Wasserstein GAN consisting of a generator and discriminator model.
175            It implements the Wasserstein loss with gradient penalty to improve the training of the
176            discriminator and generator.
177
178        5. Diffusion Models [model_type='diffusion']
179
180            Implements a diffusion process using UNet architectures for generating synthetic data.
181
182            This model integrates an autoencoder and a diffusion network, enabling both data
183            reconstruction and controlled generative modeling through Gaussian diffusion.
184
185        6. Kernel Diffusion [model_type='diffusion_kernel'] (In implementation stage)
186
187            Implements a diffusion process using UNet architectures for generating synthetic data.
188
189            This model integrates an autoencoder and a diffusion network, enabling both data
190            reconstruction and controlled generative modeling through Gaussian diffusion.
191
192        7. Copy/Paste [model_type='copy']
193
194            CopyAlgorithm is a naive machine learning model designed to generate synthetic data samples
195            for specific classes based on provided real samples. This simple approach is primarily used
196            for testing and comparison purposes, serving as a baseline method in experiments.

294    @staticmethod
295    def stratified_instance(function):
296        """
297        A decorator that performs stratified K-Fold cross-validation on the provided dataset.
298        This method shuffles the dataset, splits it into `k` stratified folds, and saves
299        the corresponding training and validation datasets as CSV files. It also logs detailed
300        information about the process and each fold.
301
302        Args:
303            function (Callable): The function to be wrapped. After completing the stratified K-Fold
304                                  cross-validation process, this function will be called with the
305                                  provided arguments.
306
307        Returns:
308            Callable: The wrapped function with stratified K-Fold processing applied.
309        """
310
311        @autoload # Custom decorator, used for auto loading data
312        def wrapper(self, *args, **kwargs):
313
314            # Track the start time of the entire stratified K-fold process
315            start_time = time.time()
316
317            logging.info(
318                "StratifiedKFold initialization started. Number of splits: %d, Shuffle: True, Random state: 42.",
319                self.arguments.number_k_folds)
320
321            try:
322                # Shuffle the data before performing stratified splitting
323                shuffled_data, shuffled_labels = shuffle(self._data_loaded, self._data_loaded_labels, random_state=42)
324
325                # Initialize the StratifiedKFold instance with the specified number of folds
326                stratified_instance = StratifiedKFold(n_splits=self.arguments.number_k_folds,
327                                                      shuffle=True, random_state=42)# RM TODO: Add parameter for random seed
328
329               
330                logging.info("Data loaded for stratification. Total samples: %d", len(self._data_loaded))
331
332                # Loop through each fold and process the stratified splits
333                for fold, (train_index, val_index) in enumerate(stratified_instance.split(shuffled_data, shuffled_labels)):
334
335                    # Track the start time of processing a specific fold
336                    fold_start_time = time.time()
337
338                    logging.info("Processing fold %d of %d.", fold + 1, self.arguments.number_k_folds)
339                    logging.info("Train indices (size %d): %s", len(train_index), train_index)
340                    logging.info("Validation indices (size %d): %s", len(val_index), val_index)
341
342                    # Concatenate the data and labels for the shuffled dataset
343                    data_with_labels = numpy.column_stack((shuffled_data, shuffled_labels))
344
345                    # Create a DataFrame for saving the shuffled dataset as CSV or XLS
346                    columns = [f"feature_{i}" for i in range(shuffled_data.shape[1])] + ["label"]
347                    data_frame = pandas.DataFrame(data_with_labels, columns=columns)
348
349                    # Save the shuffled data to CSV
350                    csv_filename = "{}/data_shuffled_dataset.csv".format(self.directory_output_data)
351                    data_frame.to_csv(csv_filename, index=False)
352
353                    # Concatenate training data and labels (X and y)
354                    data_with_labels = numpy.column_stack((self._data_loaded[train_index],
355                                                           self._data_loaded_labels[train_index]))
356
357                    # Create a DataFrame for saving the training dataset as CSV
358                    columns = [f"f{i}" for i in range(shuffled_data.shape[1])] + ["label"]
359                    data_frame = pandas.DataFrame(data_with_labels, columns=columns)
360
361                    # Save training fold data to CSV
362                    csv_filename = "{}/data_training_fold_{}.csv".format(self.directory_output_data, fold+1)
363                    data_frame.to_csv(csv_filename, index=False)
364
365                    # Concatenate validation data and labels (X and y)
366                    data_with_labels = numpy.column_stack((self._data_loaded[val_index], self._data_loaded_labels[val_index]))  
367
368                    # Create a DataFrame for saving the validation dataset as CSV
369                    columns = [f"f{i}" for i in range(shuffled_data.shape[1])] + ["label"]
370                    data_frame = pandas.DataFrame(data_with_labels, columns=columns)
371
372                    # Save validation fold data to CSV
373                    csv_filename = "{}/data_evaluation_fold_{}.csv".format(self.directory_output_data, fold+1)
374                    data_frame.to_csv(csv_filename, index=False)
375
376                    # Shuffle the training and evaluation data
377                    training_shuffled_data, training_shuffled_labels = shuffle(self._data_loaded[train_index],
378                                                                               self._data_loaded_labels[train_index], random_state=42)
379
380                    evaluation_shuffled_data, evaluation_shuffled_labels = shuffle(self._data_loaded[val_index],
381                                                                                   self._data_loaded_labels[val_index], random_state=42)
382
383                    # Store the training and evaluation data for later use
384                    self.list_folds.append({
385                        'x_training_real': training_shuffled_data,
386                        'x_evaluation_real': evaluation_shuffled_data,
387                        'y_training_real': training_shuffled_labels,
388                        'y_evaluation_real': evaluation_shuffled_labels,
389                        'x_evaluation_synthetic': None,
390                        'y_evaluation_synthetic': None
391                    })
392
393                    # Track the end time for this fold
394                    fold_end_time = time.time()
395
396                    logging.info("Fold %d processed successfully in %.2f seconds.", fold + 1,
397                                 fold_end_time - fold_start_time)
398
399                # Track the end time for the entire K-Fold process
400                end_time = time.time()
401
402                logging.info("All %d folds successfully created in %.2f seconds.", self.arguments.number_k_folds,
403                             end_time - start_time)
404
405                # Call the original function passed as a decorator argument
406                return function(self, *args, **kwargs)
407
408            except Exception as e:
409                logging.error("An error occurred during StratifiedKFold processing: %s", str(e))
410                raise # Re-raise the exception after logging the error
411
412        return wrapper

416    def evaluation_TS_AR(self, dictionary_data):
417
418        """
419        Evaluates the performance of classifiers trained on synthetic data and evaluated on real data.
420        This method trains classifiers using synthetic data (generated data) and evaluates them using
421        real evaluation data. The binary classification metrics for each classifier are calculated and
422        logged for each fold.
423
424        Args:
425            dictionary_data (dict): A dictionary containing real evaluation data. The key 'x_evaluation_real'
426                                    holds the features for evaluation, and the key 'y_evaluation_real' holds the true labels.
427        """
428        # Logging the evaluation strategy
429        logging.info(f"\tTS-ER: train on synthetic, evaluate on real")
430
431        # Initialize empty lists for labels and data
432        labels, data = [], []
433
434        # Logging the total number of generated synthetic samples to be processed
435        total_samples = sum(len(samples) for samples in self.data_generated.values())
436        logging.info(f"\t\tTS-ER: Total number of samples to be saved: {total_samples}")
437
438        # Iterate through each class and its corresponding generated synthetic samples
439        for label_class, generated_samples in self.data_generated.items():
440
441            logging.info(f"\t\tTS-ER: Processing {len(generated_samples)} samples for label class {label_class}.")
442
443            # Add the label (class) for each generated sample
444            labels.extend([label_class] * len(generated_samples))
445
446            # Add generated samples to the data list
447            data.extend(generated_samples)
448
449        # Train classifiers using the generated synthetic data and corresponding labels
450        classifiers = self.get_trained_classifiers(data, numpy.array(labels), numpy.float32, self.get_number_columns())
451
452        # Evaluate the classifiers on real data for each classifier instance
453        for classifier_name, classifier_instances in zip(self._dictionary_classifiers_name, classifiers):
454
455            # Predict the labels using the trained classifier on the real evaluation data
456            label_predicted = classifier_instances.predict(dictionary_data['x_evaluation_real'])
457                
458            # Calculate and log the binary classification metrics (such as accuracy, precision, recall, etc.)
459            self.get_binary_metrics(numpy.squeeze(dictionary_data['y_evaluation_real'], axis=-1)[:total_samples],
460                                        numpy.array(label_predicted)[:total_samples],
461                                    "TS-TR", classifier_name, self.fold_number+1)
